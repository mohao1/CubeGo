# Golang常用工具包

## 介绍
Golang常用数据扩展工具包。

## 功能设计

**模块设计**
1. Heap 堆
- 大顶堆：maxHeap。
- 小顶堆：minHeap。
2. Stack 栈
- Stack：顺序栈。
- LinkStack：链表栈
3. Queue 队列
- Queue：顺序队列
- LinkQueue：链表队列
4. Util 工具
- Compare：选择二数选一，选择大/小。
- Equal：数据比较，比较数据大小。
5. ExecutorsPool：线程
- Executors：线程池
6. List：链表
- LinkList：单向链表
- BLinkList：双向链表
